# Role-Based Auth System

A React-based authentication application demonstrating role-based access control (RBAC) with separate user and admin dashboards. Built with React Router and Context API for state management.

## Features
- User registration and login
- Role-based access (User/Admin)
- Protected routes with authentication checks
- Persistent sessions using localStorage
- Clean, responsive UI
- Form validation with error handling

## Tech Stack
- React 18
- React Router v6
- Context API for state management
- LocalStorage for data persistence
- CSS3 for styling
